# Refinitiv Data Explorer

## 概要
スプレッド予測精度向上のための主要データアクセス可能性検証プロジェクト

## 検証対象データカテゴリー

### 1. リアルタイムの需給ファンダメンタルズ
- 在庫データ（公表・非公表）
- 生産・供給データ
- 物流・輸送データ

### 2. フォワードルッキングなデータ
- 現物プレミアム
- アナリストレポートと需給予測
- 川下産業のデータ

### 3. 市場参加者のポジションとセンチメント
- 投機筋・実需筋のポジション
- オプション市場のデータ

### 4. 金利と資金調達コスト
- 短期金利とフォワードカーブ

## 注意事項
このプロジェクトは既存のLMEレポートシステムとは完全に独立しており、既存コードに影響を与えません。
